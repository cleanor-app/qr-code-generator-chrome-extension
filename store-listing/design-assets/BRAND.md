# Cleanor QR — brand tokens (for the store graphics)

## Color
- Primary blue: `#4576FD`
- Blue gradient (hero backgrounds): radial `#5A83FF` (center) → `#4576FD` (edge)
- Brand-strong: `#2F55D4`
- Ink (headlines on light): `#1C2434`
- Muted body text: `#5B6B86`
- Light surfaces: `#FFFFFF`, `#F6F8FC`, `#EEF3FF`
- Light-blue text on blue: `#D7E2FF`
- Dark surface (the "private" slide): gradient `#0F1524` → `#1B2740`; text `#EEF3FF` / muted `#9FB0D0`

## Type
- Headline: elegant serif — Newsreader / Apple "New York", weight 600–700, tight tracking.
- Body / UI: clean sans — SF Pro Display / Helvetica Neue, weight 500.

## Feel
- Generous whitespace, large soft shadows, rounded cards (radius ~24px), premium and calm.
- The product UI is always the hero; text supports it, never crowds it.

## Voice
- Restrained and concrete. Benefit-led, no keyword stuffing, no "#1 / best" claims, no invented metrics.
